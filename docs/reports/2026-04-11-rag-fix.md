# Rapport — Fix RAG vectoriel (2026-04-11)

## Contexte

Depuis la Phase 3a, le RAG vectoriel (`app/embedding.py`) était inactif en production.
Cause : le package `openai` n'était pas dans `requirements.txt`.
Impact : `is_available()` retournait `False`, toutes les fonctions de recherche sémantique
(`retrieve_context`, `search_similar`, `validate_rule_before_save`) tombaient en fallback non-vectoriel.
